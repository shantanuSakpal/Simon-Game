Press Any Key to Start

How To Play

  1. Simon will give the first signal. Repeat the signal by pressing the same color button.
  2. Simon will give one more signal. Repeat the signals by pressing the same color button, in order.
  3. Simon will give one more signal. Repeat the signals by pressing the same color button, in order.
  4. Continue playing as long as you can repeat each sequence of signals correctly.
  5. If you fail to repeat a sequence exactly, Simon responds with a "RAZZ" sound. This means you've lost, and the sequence of signals ends.
